# 🏗 架构决策记录 (ADR)

> 重大技术决策的记录。每条决策一旦做出，不应轻易推翻。

---

## ADR 格式

```
## ADR-{序号}: {决策标题}

- **日期**: YYYY-MM-DD
- **决策者**: WorkBuddy / Reasonix
- **状态**: 提议 / 已接受 / 已废弃

### 背景

（为什么需要做出这个决策？面临什么问题？）

### 决策

（我们选择了什么方案？）

### 备选方案

1. 方案A — （优缺点）
2. 方案B — （优缺点）

### 影响

（这个决策对项目有什么影响？需要做什么变更？）

---
```
